•字符串查找 

​	–indexOf (String str)

是否包含某一字串，返回字串下标

​	–lastIndexOf(String str)

找到字串，找到字串最后出现的位置

```java
String str7 = "abcabc";
System.out.println(str7.indexOf("bc"));//1
System.out.println(str7.lastIndexOf("bc"));//4
```
